◆ SilKit_RpcClient_CallWithTimeout()

SilKit_ReturnCode SilKit_RpcClient_CallWithTimeout ( SilKit_RpcClient *  self,
const SilKit_ByteVector *  argumentData,
SilKit_NanosecondsTime  timeout,
void *  userContext 
)

Initiate a remote procedure call with a specified timeout.

In a synchronized execution, simulation time is used for the timeout, in a unsynchronized execution, system time is used for the timeout.

Parameters
selfThe RPC client that should trigger the remote procedure call.
argumentDataA non-owning reference to an opaque block of raw data
timeoutA duration in nanoseconds after which the call runs into a timeout and the CallResultHandler is called with status timeout. After the timeout occurred, no further call result events will be triggered for this call.
userContextAn optional user provided pointer that is reobtained when receiving the call result.
